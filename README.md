# FreshMart 3D — live demo

**▶ Try it: https://tpminhtam.github.io/freshmart-3d/**

A browser-based 3D grocery store built with Three.js (r185). Drive the cart with the arrow keys, press **Space** to grab the nearest shelf item, and chat with Mr. Peel, the produce-aisle assistant.

## Highlights

- 175-product catalog rendered as archetype packaging geometry (cans, cartons, boxes, tubs…) with procedurally wrapped label art
- Real photogrammetry scans (YCB, Google Scanned Objects, Poly Haven) alongside the generated packaging
- HDRI image-based lighting, GTAO, bloom, and filmic tone mapping

This repository holds the deployed runtime of the demo; development (rendering pipeline, catalog/label tooling, research notes) happens in a private repo.
